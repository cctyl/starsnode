
# 目标
制作一个手机端使用的监控平台

# 数据格式
data.json 就是实际数据

# 要求

你需要制作一个html页面，把data.json中的数据完整的解析出来。
先出一个原型图用html来展示。


## 布局
因为是提供给手机端查看的，所以字体、缩放等要按照手机的标准设计

## 汇总
页面的最顶端，需要一个汇总，把所有设备的信息汇总在一起，方便查看。
具体来说，有这些：
- 总在线设备数
    - win系统在线设备数
    - linux系统在线设备数
    - 其他系统在线设备数
- 设备硬件汇总信息
    - CPU核心数汇总
    - cpu平均使用率
    - 磁盘总大小
    - 内存总大小

最好用图表展示


## 单个设备详情

需要首先需要展示设备的基本信息，cpu，内存，硬盘，以及该设备当前网络总上传下载速度，最好能用图表展示。
然后需要罗列该设备每个网卡信息。
然后要显示设备每个网卡的网络上传下载速度。
